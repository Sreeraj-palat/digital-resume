DeveSearch - Developer search Platform

  • Source code

Developers can add their portfolio and skills they have in the platform. A job higher can post job posts and interested candidates can send an interest to the job post. Users can also chat each other through the platform and once they committed a work the job higher can make payment to the developer through the website.

  • Software and Technologies Used :-
  • Backend: Django
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Database: PostgreSQL
  • API Integrated: RazorPay
