Ashion - Ecommerce Platform

  • Source code

Currently working on an ecommerce website using django

Website that allows people to buy Dresses, Cosmetics, and Accessory products over the internet. Through this website, a business can process orders, accept payments, manage shipping and logistics, and provide customer service.

  • Backend: Django
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Database: PostgreSQL
  • API Integrated: Twilio, RazorPay
  • Users can access the platform as a guest, users can add products to cart
  • Review and rate purchased products.
  • Admin login, block user, add and edit product, add and edit category, track payments, order status
